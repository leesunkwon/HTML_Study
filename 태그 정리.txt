<head> 태그
웹 사이트에 대한 정보 외부 자료 참고 <CSS, JS>

<body> 태그
실제 화면의 내용

태그 사용 방법
<태그>내용</태그>

<title> 제목 </title> (웹사이트 이름)

<body>에 들어가는 태그

H1 ~ H6 태그 :  주로 글의 제목이나 강조를 위해 사용하는 태그
사용 예시 : <h1>내용</h1>

P 태그 : 문단을 넣을때 주로 사용
사용 예시 : <p>내용</p>s

INPUT 태그 : 입력창을 의미하며 내용이 없기 때문에 닫는 태그가 불필요함
사용 예시 : <input type="text">
type = 해당 태그의 속성
type="text" = type 속성의 값 = text

BUTTON 태그 : 버튼
사용 예시 : <button>버튼</button>

DIV 태그 : 여러개의 태그를 구역으로 묶어 나눌때 사용

BR 태그 : 텍스트 줄바꿈
사용 예시 : <p>안녕하세요.<br/>반갑습니다.</p>
결과
안녕하세요.
반갑습니다.
